# FishTouchingCoin

摸鱼币是一款常驻桌面的摸鱼收益计时器，帮助你在工作日/周末、午休、锁屏等场景下自动统计摸鱼收入，并在托盘中低打扰运行。

## 功能亮点
- 悬浮置顶小窗口，实时展示当前收益与状态。
- 午休/下班时间自动停算，周末支持倍率。
- 右键菜单支持详情趋势、重新配置、暂停计费与退出。
- 托盘化运行：最小化后隐藏任务栏入口，仅保留托盘图标。
- 自动落盘与历史记录保留策略，避免高频写盘。

## 快速开始
```bash
pip install -r requirements.txt
python app.py
```

首次启动会弹出配置窗口，录入月薪、工作时间等参数即可开始计算。

## 使用提示
- 老板键默认 `F9`，用于快速隐藏/显示窗口。
- 工作日 09:00 前不计入收益；午休时间与下班时间自动停算。
- 锁屏在允许带薪时长内仍可计费，超过阈值后暂停。

## 数据与配置位置
应用会将数据和配置写入用户本地数据目录：
- Windows：`%APPDATA%\FishTouchingCoin`
- 其他平台：`~/.local/share/FishTouchingCoin`（或 `XDG_DATA_HOME`）

目录内包含：
- `data_schema_v*.json`：每日收益与历史记录
- `settings_schema_v*.json`：个性化配置
